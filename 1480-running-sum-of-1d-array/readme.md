## 1480. Running Sum of 1d Array
https://leetcode.com/problems/running-sum-of-1d-array/description/

Time complexity: O(n) because the number of operations is the same for each element.
n is the number of elements in the input array.

Space complexity: O(n) because we declare an array result of size n to store the running sum and access the previous value to calculate the next value.
